# 九轉序列

- [指標參考影片來源](https://www.youtube.com/watch?v=gd78FyjyY04&ab_channel=2140%E5%8A%A0%E5%AF%86%E7%A4%BE%E7%BE%A4)

說明
---

**單邊行情連續9根K棒(13根)收盤價大於(小於)該K棒的前四根，極高機率出反轉(參考圖1)**
* 從第7根開始標示至第9根K棒
* 從第11根開始標示至第13根K棒

<br/>

**設定(參考圖2)**
* 可勾選是否顯示該指標
* 可選擇顯示 **9**、**13** 或者 **9+13**
